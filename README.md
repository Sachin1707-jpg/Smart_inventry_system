# Smart_inventry_system
# 🚀 Smart Inventory Management System for SMEs

## 📌 Overview

Small and Medium Enterprises (SMEs) often rely on manual methods like Excel sheets or paper-based systems to manage inventory, orders, and manufacturing processes. This leads to errors, lack of real-time tracking, and inefficient decision-making.

This project provides a **centralized, web-based solution** to automate inventory management, streamline order processing, and track manufacturing workflows in real-time.

---

## 🎯 Problem Statement

* Manual inventory tracking leads to **errors and inconsistencies**
* No real-time visibility of stock levels
* Difficult to manage sales, purchases, and manufacturing together
* Poor decision-making due to lack of insights

---

## 💡 Proposed Solution

We developed a **Smart Inventory Management System** that:

* Tracks inventory in real-time
* Manages sales and purchase orders
* Monitors manufacturing processes (Work-In-Progress)
* Provides a dashboard with actionable insights

---

## ⚙️ Features

### 📦 Inventory Management

* Add, edit, delete products
* Track stock levels
* Low stock alerts

### 📑 Order Processing

* Create sales & purchase orders
* Automatic stock updates

### 🏭 Manufacturing (WIP Tracking)

* Track production stages
* Manage raw material usage
* Status tracking (Pending / In Progress / Completed)

### 📊 Dashboard & Reports

* Real-time data visualization
* Summary cards (Total Products, Orders, Low Stock)
* Basic charts for insights

---

## 🏗️ System Architecture

The system follows a **3-tier architecture**:

### 1. Frontend (Client Side)

* Built using React / HTML-CSS-JS
* Provides user interface and dashboard
* Sends API requests to backend

### 2. Backend (Server Side)

* Built using Node.js / Flask
* Handles business logic and API requests
* Manages authentication and data processing

### 3. Database (Storage Layer)

* MongoDB / MySQL
* Stores product, order, supplier, and manufacturing data

### 🔄 Data Flow

User Action → Frontend → Backend API → Database → Backend → Frontend (Updated UI)

---

## 🔄 Workflow

1. Add product to inventory
2. Create sales/purchase order
3. Stock automatically updates
4. Manufacturing process starts (if required)
5. Dashboard updates in real-time

---

## 🛠️ Tech Stack

* **Frontend:** React / HTML, CSS, JavaScript
* **Backend:** Node.js (Express) / Flask
* **Database:** MongoDB / MySQL
